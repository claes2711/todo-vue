# todo-vue

To run the app, you will need to 
1. open terminal
2. install homebrew
command : /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
3. install git by brew
command : brew install git
4. cd into todo-vue
5. install nvm
command : curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.1/install.sh | bash
6. add nvm to bash profile
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"
7. install node
command : nvm install node
8. install npm package
command : npm install --global or npm i
9. now you would be able to run the app on server
command : npm run dev
